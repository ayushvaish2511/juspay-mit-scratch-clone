# Project

- This is basically a clone of the scratch site by MIT it is a site where you can create your own custom games and animations.


# How to run the project ?

- Clone the project
- Run `npm install`
- Run `npm start`
- Go to `localhost:8080`


# Tech stack used

- React
- Redux
- React DnD 
- React Redux
- React Redux Toolkit
- Tailwind CSS
